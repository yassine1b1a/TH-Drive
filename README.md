# TH-Drive - Ride Sharing Platform

A modern ride-sharing platform built with Next.js 14, Supabase, and Tailwind CSS.

## Features

- **User Authentication**: Secure login/signup with email verification
- **Ride Booking**: Real-time ride booking with interactive maps
- **Driver Management**: Driver registration, verification, and ride management
- **Real-time Tracking**: Live location tracking for rides
- **Payment Integration**: Multiple payment methods support
- **Rating System**: Driver and rider rating system
- **Admin Dashboard**: Moderation and management tools

## Tech Stack

- **Frontend**: Next.js 14 (App Router), React, TypeScript
- **Styling**: Tailwind CSS, shadcn/ui
- **Database**: Supabase (PostgreSQL)
- **Authentication**: Supabase Auth
- **Maps**: Leaflet.js, OpenStreetMap
- **Deployment**: Vercel
